rawmk
=====

RAW photo workflow script based on GNU Make

Features:

- multi-image workflows - panorama, exposure and focus stacking etc.
- stereoscopic image sorting and alignment
- sorting files into directories
- simple processing 


Requires:

RawTherapee
Hugin
Panotools::Script
exiftool
...
